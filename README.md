# Portfolio V2

Minimal developer portfolio built with React, TypeScript, and Tailwind.

## Stack

-   React + TypeScript
-   Tailwind CSS
-   React Router
-   Firebase (contact form)

## Run locally

``` bash
git clone https://github.com/Kiuliumov/portfolioV2.git
cd portfolioV2
npm install
npm run dev
```
